《乌合之众》读书笔记

Posted on 2018-05-20 | Edited on 2018-08-23 | In 读书笔记 | Comments: | Views:

作者勒庞

乌合之众定义：

同一时间空间，一群无组织，彼此不想干的人，因为某个事件聚集起来，这群人因为事件的发生具备心理上的共通点。

心理特点:

1.本能优先：逃生、破坏、同情

2.互相传染：群体行动的传染性

3.易受暗示

4.缺乏理性，个人智商加入群体，会智商下滑，个体情绪频率和群体频率配对，情绪配对，个人放弃独立思考，容易夸张。

好的方面：

群里的救人、捐助行为

历史事件

  • 法国大革命：议税会议，国王像第三等级征税，群众不满

  • 义和团等历史运动

乌合之众形成的直接因素：

  • 模糊概念词汇 自由平等博爱等模糊词汇

  • 集体行动中的幻觉，虚构目标、场景、运动中幻觉更容易出现，置身山呼海啸的运动中，他们会觉得他们是不可阻挡的，革命胜利就在前方

  • 群体具备的历史经验，法国国王之前向第三等级征税很多次了，旱灾等政府救灾不利，共同的生活经验产生不满

乌合之众形成的间接因素：

  • 种族：种族决定了一群人的气质，法国人的气质是容易冲动、容易受到蛊惑的民族，英国人是理性沉稳，不像法国人这么偏执，容易受到暗示，同样改朝换代，英国人搞的光荣革命基本没有死人，法国人攻占了巴士底狱，把路易十六送上了断头台。

  • 传统因素：和种族是绑定的，这一点上，勒庞和鲁迅很像，不同国家不同种族的人是有各自特点气质的，鲁迅批判中国人哀其不幸怒其不争，勒庞批判法国人太冒进不够理性

  • 时间：传统、政治制度、教育都要依靠时间的积累，再牛的人要做成一件事都需要时间，等待能量储备。

  • 政治、社会制度：同样民主制度，英国美国高度繁荣，拉丁民族就实现的比较扯（西班牙殖民过的阿根廷、智力玻利维亚很混乱）

  • 教育：法国教育是把受过教育的人变成敌人

小结一下：种族是核心因素

领袖在乌合之众的作用：

一时拥有领袖气质的人，临时性暂时性的领袖，球迷骚乱的先锋，跳水救人的见义勇为者，做那件事的时候成为领袖。

拥有更持久的意志力，长期占据主要位置，死后的影响可能超过生前，革命领袖、开国元勋、大思想家、传奇商人

领袖的手段：

do {

断言：坚定地下判断，没有论证，坚定的语气能让台下的观众觉得正常和舒适，说理和论证战胜不了一些词语和套话，只要一听到他们人人都会肃然起敬，扶手昂立，大而化之的套话振奋人心，精神抖擞。

重复：造成人心浮动，拿破仑说极为重要的修辞法只有一个那就是重复，拿破仑是鼓动专家。

传染：日本福岛核泄大家抢盐，是因为有人说吃盐可以低于辐射，在一个群体中恐惧会让人不管不顾，谣言止于智者，都陷入恐慌就不存在智者

通过舆论、文学和艺术等等进行体现，达到个人崇拜的目的。

} while(get 名望)

如何看待这本书的意义：

仰视社会学家的洞见、平时他的论述，鸟瞰他的观点在整个思想脉络中的位置，知道他的位置才能作出批判性的解读，才能理论联系实践，乌合之众的当代意义是他发现问题的功能，而非解决问题的功能，他下的判断很精准，他描述的群众心里特点发人深省、让我们避免被某个夸大宣传的产品所蛊惑，不自觉被某个大V的言论带领，成为乌合之众的门槛很低，人类有一种服从权威命令的天性，在一定情境下，人们会遵照权威的说辞放下道德约束去伤害无辜的人，当然也有在集体行动中寻求个人利益的人，只是搭便车，比如混进游行中以便领盒饭的人。
